Добавьте сюда файл уведомления Telegram в формате MP3 с именем `telegram-notify.mp3`.

Рекомендуемое расположение: `src/audio/telegram-notify.mp3`

Если файл присутствует, виджет будет пытаться использовать его при уведомлениях. Если файл отсутствует или загрузка не удалась, виджет использует встроенный WebAudio fallback (короткие тоны).
